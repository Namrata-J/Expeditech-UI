Avatar

An avatar component is a profile image often used in combo with additional elements (e.g., a status element) to provide additional info about a user.

Expeditech UI provides user with varities of avatars i.e. circular avatar, rounded avatar, circular avatar with top notification, circular avatar with bottom notification, rounded avatar with top notification, rounded avatar with bottom notification.

Circular avatars

Expeditech UI provides user with all sizes of circular avatars i.e. x-small, small, medium and large.

Circular avatars

Copy & paste the below html template to use these components in your projects.

For changing the size of the circular avatar pick the pre-defined classes for avatar component, for example say ".xs-avatar"class and apply "width" and "height" property on it.

Circular avatars with top notification

Expeditech UI provides user with all sizes of circular avatars with top notification i.e. x-small, small, medium and large. The notificatons provided are in three colors i.e. red, green, gray. These notification colors can be customized according to user requirements.

For changing the position of top notification pick the pre-defined classes for avatar component, for example say pick ".et_notifi-avatar-n"class and apply "top" and "right" property on it.
