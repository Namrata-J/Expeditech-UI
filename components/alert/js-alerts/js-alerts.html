Alerts

Alerts are urgent interruptions, requiring acknowledgment, that inform the user about a situation.

Js alerts

Expeditech UI provides you alerts for any length of text, as well as an optional dismiss button with js functionality. It also allows the user to customize the components according to their specifications.

Expeditech UI provides four varities of js alerts i.e. dismissible simple alerts, dismissible simple alerts with strong color, dismissible simple alerts with icon, advanced dismissible alerts.

Dismissible simple alert is an alert with a optional dismiss.
