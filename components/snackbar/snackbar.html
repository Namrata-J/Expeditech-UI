Snackbars

Snackbars inform users of a process that an app has performed or will perform. They appear temporarily, towards the bottom of the screen. They shouldn't interrupt the user experience, and they don't require user input to disappear.

Or in simple words, snackbars communicate messages that are minimally interruptive and don't require user action.

Expeditech UI provides dismissible snackbars.

Expeditech UI provides varities of snackbars i.e. baseline snackbar, leading snackbar, stacked snackbar.
