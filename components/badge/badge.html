Badges

The badge component can be used to complement other elements such as buttons or text elements as a label or to show the count of a given data, such as the number of comments for an article or how much time has passed by since a comment has been made.

Expeditech UI provides you badges with customizable pre-defined classes.

Expeditech UI provides you varities of badges i.e. badges on buttons, badges in buttons, badges on avatars, badges on icons.

Badges on buttons

Expeditech UI provides user with varities of badges on buttons i.e. x-small circular badges on buttons, small circular badges on buttons, medium circular badges on buttons, x-small rounded badges on buttons, small rounded badges on buttons, medium rounded badges on buttons, x-small circular toggle badges on buttons.
