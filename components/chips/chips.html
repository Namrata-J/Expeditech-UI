Chips

Chips are compact elements that represent an input, attribute, or action. Chips allow users to enter information, make selections, filter content, or trigger actions.

Expeditech UI provides varities of chips i.e. automated chips, drag and drop chips, basic chips, stacked chips.

Automated chips

To see how these chips work, click on the "favourite items". You will observe that input field appears. Then you can enter any counts of inputs and can discard them as well. Besides, when focus is lost then the "favourite item" returns back to its original state.
