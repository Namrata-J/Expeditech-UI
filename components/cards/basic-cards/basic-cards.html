Cards

Cards contain content and actions about a single subject. Cards are surfaces that display content and actions on a single topic. They are easy to scan for relevant and actionable information.

Basic Cards

Expeditech UI provides user with varities of basic cards, i.e. basic card, basic w/ text over media, basic w/ header, basic w/ only action buttons, basic w/ only icons, basic w/ horizontal image and text.

Expeditech UI provides user with same html template for all the basic cards, by just altering the contextual classes.

All basic cards template comprises four sections, wherein the first section comprise image, the second section comprise heading and subheading, the third section comprises the para, and the fourth section contains the action buttons and the icon buttons.

PS: The template for all the basic cards is same, the only difference is with the styling.

Basic card shows the basic template keeping the visibility of all the sections in order.
