Utilities

Expeditech UI provides user with quite a bunch of utility classes, using which you can get rid of writing extra css code.

Text utility

Expeditech UI provides varities of text utilities i.e. headings, texts, gray-texts, aligned texts, weighted-texts..

Headings

Expeditech UI provides utilities for headings, you can simply assign header tag and it will assign predefined line height and font-size to them automatically.
